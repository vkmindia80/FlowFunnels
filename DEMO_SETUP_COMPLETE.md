## How to Test the Drag-and-Drop Editor

### Step 1: Login
1. Go to the login page
2. Enter email: `demo@flowfunnels.com`
3. Enter password: `demo123`
4. Click "Sign In"

### Step 2: Access Funnel Builder
1. You'll see the Dashboard with "Demo Funnel"
2. Click the **"Edit"** button on Demo Funnel
3. You'll be taken to the funnel builder

### Step 3: Create Your First Page
1. Click **"Add Page"** button
2. Enter a page name (e.g., "Landing Page")
3. Click "Add Page"
4. The page will open in the editor

### Step 4: Build Your Page with Drag-and-Drop

#### Add a Heading
1. **Drag** the "Text" element from the left sidebar
2. **Drop** it onto the canvas (white area)
3. **Click** on the text element to select it
4. In the **right panel**, edit:
   - Text: "Welcome to FlowFunnels!"
   - Tag: "Heading 1"
   - Font Size: "48px"
   - Color: Click color picker to change

#### Add a Button
1. **Drag** the "Button" element to canvas
2. **Drop** it below the heading
3. **Click** to select
4. Edit in right panel:
   - Button Text: "Get Started"
   - Link URL: "#signup"
   - Background Color: Blue (#0ea5e9)
   - Border Radius: "12px"

#### Add an Image
1. **Drag** "Image" element to canvas
2. **Drop** it in desired position
3. **Click** to select
4. Edit Image URL: `https://via.placeholder.com/600x400`
5. Adjust width and height as needed

#### Add a Form
1. **Drag** "Form" element to canvas
2. **Drop** it on the page
3. Click to select and customize

#### Add a Video
1. **Drag** "Video" element to canvas
2. **Drop** it on the page
3. Paste YouTube embed URL (e.g., `https://www.youtube.com/embed/dQw4w9WgXcQ`)

### Step 5: Use Canvas Tools

#### View Modes
- Click **Monitor icon** - Desktop view (1280px)
- Click **Tablet icon** - Tablet view (768px)
- Click **Smartphone icon** - Mobile view (375px)

#### Zoom Controls
- Click **-** to zoom out
- Click **+** to zoom in
- Click **Maximize** to reset to 100%

#### Editing Elements
- **Click** any element to select it (blue ring appears)
- Use **right panel** to edit properties
- Change position by editing X, Y values
- Change size by editing width, height
- Click **X icon** in right panel to delete

### Step 6: Auto-Save
- Changes save automatically every 2 seconds
- Watch the save status in top-right:
  - "Saving..." - Currently saving
  - "Saved" (green check) - All changes saved
  - "Unsaved changes" - Waiting to save
- Click **"Save"** button to save immediately

### Step 7: Multiple Pages
1. Click **"Add Page"** to create another page
2. Switch between pages using **tabs** below header
3. Each page has its own elements

## Tips for Best Results

### 1. Element Positioning
- Elements drop where you release the mouse
- Use X/Y coordinates in properties panel for precise positioning
- Canvas has grid layout for alignment

### 2. Element Sizing
- Each element type has default size
- Adjust width/height in properties panel
- Consider responsive design for different views

### 3. Content Editing
- Click element to see its properties
- Different element types have different options
- Text: font size, color, tag (h1/h2/h3/p)
- Button: text, link, colors, border radius
- Image: URL, alt text
- Form: field types, submit button text
- Video: embed URL

### 4. Saving
- Auto-save works every 2 seconds
- Manual save if you want immediate save
- Watch save status indicator

### 5. Multi-Page Workflow
- Create landing page first
- Then create thank you page
- Use tabs to navigate between pages
- Each page saves independently

## Troubleshooting

### Login Issues
**Problem:** "Incorrect email or password"
**Solution:** Demo user has been created. Use: demo@flowfunnels.com / demo123

### Canvas Not Loading
**Problem:** Blank canvas or not responding
**Solution:** Refresh the page, ensure you have a page created

### Elements Not Saving
**Problem:** Changes don't persist after reload
**Solution:** Wait for "Saved" indicator, check browser console for errors

### Drag Not Working
**Problem:** Can't drag elements
**Solution:** Make sure you're dragging from the library (left sidebar), not from canvas
